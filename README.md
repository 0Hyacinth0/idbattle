# id大作战 (ID Battle)

这是一个基于纯前端的自动战斗模拟与角色养成（配装）系统。项目经过了彻底的现代化重构，采用 Vite 构建，实现了底层战斗逻辑与 UI 的完全解耦，并引入了单向数据流进行状态管理。

## 🌟 核心特性

- **高度解耦的战斗引擎**：战斗核心（回合管理、伤害计算、技能触发、状态效果）已从 UI 抽离，支持独立的 Node.js 环境下运行大规模回归与平衡性模拟。
- **现代化事件流架构**：采用自建 `EventBus`，引擎逻辑不再直接操作 DOM，而是广播标准化事件。
- **响应式状态管理**：内置轻量级 `BattleStore`，所有 UI 组件（日志面板、玩家血量条、属性面板）均通过订阅 Store 响应状态变化，保证了数据一致性。
- **配装定制系统 (Growth Lab)**：允许玩家自由分配属性点，装备不同品质的武器防具，并选择特殊技能，所有定制数据支持无缝代入战斗流。

## 🛠 环境准备

- Node.js 20.19+ 或 22.12+ (Vite 依赖要求)
- npm 9+

克隆项目后首先安装依赖项：

```bash
npm install
```

## 🚀 运行项目

项目目前基于 **Vite** 提供热重载（HMR）开发服务器与生产级打包：

### 启动开发服务器
```bash
npm run dev
```
运行后，在浏览器访问 `http://localhost:3000` 即可实时预览并开发。

### 构建生产版本
```bash
npm run build
```
这会在 `dist` 目录下生成高度优化的前端构建产物（HTML/CSS/JS）。

## 🧪 测试与模拟

项目中包含完备的测试脚本和模拟工具，用于确保战斗机制的稳定。

### 运行单元测试
```bash
npm run test
```
使用 Node.js 原生的 `--test` 模块对核心引擎的数值逻辑、软上限修正、装备加成等进行底层冒烟测试。

### 批量战斗平衡分析
可以使用内置的 `balance-analysis.js` 脚本在 Node 环境快速执行上千次回合模拟，并在控制台输出可视化图表与胜率指标：

```bash
npm run analyze -- -a "Alice" -b "Bob" -n 1000
```

**常用参数说明：**
- `-a` / `--player1 <名字>`：指定玩家一
- `-b` / `--player2 <名字>`：指定玩家二
- `-n` / `--iterations <次数>`：对决重复次数（默认 1000）
- `--seed <随机种子>`：确保测试结果的完全可复现性
- `--json`：以标准 JSON 格式输出分析报告，可用于下游自动化数据处理流水线
- `--config`：注入特定角色的自定义配装和详细属性分配文件
